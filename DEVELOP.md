# develop

### run out of tmp directory
`gulp watch`  
build tmp dir and watch all demo/src files, run the demo out of tmp  

`gulp build-tmp`   
builds tmp dir  
  
`gulp build-dist`   
build dist dir, which has the minified files  

`gulp build-ghpages`   
builds the gh-pages branch for the github.io site
